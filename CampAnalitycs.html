Objetives


The present undergraduate thesis introduces CampAnalytics, a digital solution for camp management. The platform focuses on digitizing and streamlining inventory management, camper management, monitoring team management, and game suggestions based on the weather. To achieve this, an online application has been developed, allowing users to have a complete overview of all relevant aspects of the camp, from space reservations to activity planning and resource management. Through the application, camp managers will be able to access detailed information about available inventory, the number of campers and monitors, as well as recommendations for activities and games that are suitable for current weather conditions.

The mobile app CampAnalytics is justified by the need to improve the management and optimization of camps, addressing challenges such as attendance management, activity planning, and inventory management. By providing an efficient digital platform for camp management, CampAnalytics enhances the experience for counselors and enables organizers to carry out management processes more effectively and efficiently.

Technologies

-Kotlin: as programming language

-Android Studio IDE: as IDE

-SQL Server: as Database hosting

-JTDS Driver: as a connection between SQL Server and the application

-Android Studio ORM: used to help with the mapping
